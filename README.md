# Portfolio Website - Levon

Meine erste Website gebaut mit HTML, CSS, JavaScript und JSON.

## 📁 Struktur

```
first-project/
├── index.html       # Hauptseite
├── css/
│   └── style.css    # Styles
├── js/
│   └── script.js    # JavaScript
├── data/
│   └── data.json    # Projektdaten
└── README.md        # Diese Datei
```

## 🚀 Features

- Responsives Design
- Projekte aus JSON laden
- Smooth Scrolling Navigation
- Modern CSS mit Gradient

## 🌍 Gehostet auf

**GitHub Pages:** https://levon-levvt.github.io/first-project/  
**Custom Domain:** https://levon.levvt.com/

## 📝 Verwendete Technologien

- HTML5
- CSS3
- Vanilla JavaScript
- JSON

## 🔧 Anleitung

1. Repository klonen
2. `index.html` im Browser öffnen
3. Projekte in `data/data.json` bearbeiten
